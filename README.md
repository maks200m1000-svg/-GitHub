# GitHub User Finder
Автор
Максим Гриценко

GitHub User Finder - это графическое приложение для поиска пользователей на платформе GitHub. Приложение позволяет искать пользователей, просматривать их основную информацию, добавлять в избранное и сохранять список избранных пользователей локально в формате JSON.

- 🔍 Поиск пользователей GitHub по логину
- 📋 Отображение списка найденных пользователей
- ⭐ Добавление пользователей в избранное
- 💾 Автоматическое сохранение избранных в JSON файл
- ❌ Удаление пользователей из избранного
- 👤 Просмотр подробной информации о пользователе
- 🌐 Открытие профиля пользователя в браузере
- 📁 Хранение данных между сессиями
